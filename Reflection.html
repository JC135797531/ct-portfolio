Reflection

Over the past year in Computing Tech I have learnt heaps on building and styling websites.
At the start of the year we were introduced to html and css,
which at first seemed completely foreign but we later built up knowledge on how the two work hand in hand.
We started by installing Visual Studio Code, which made life much easier with prebuilt in code that you can type with a single shortcut.
Inside of VSC, we started with a basic html boilerplate to learn the basics of adding fonts and headers and text. We then learnt the basics of css,
allowing us to style our websites with font sizes, spacing, flexboxes, and many others. Nearing the end of the term, we starting working on our official
website portfolios through Github repositories stored using Git. After finally finishing my html with headers, footers, boxes, navigation bars and many more
I uploaded it with commits to Github and made the website live.
Overall, I have greatly enjoyed this subject so far and loved building my own, unique website and letting others see it as well.
